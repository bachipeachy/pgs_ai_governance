What is the Collatz Conjecture?

Pick any positive integer. Apply one simple rule repeatedly:

If n is even → n ÷ 2   •   If n is odd → 3n + 1

The conjecture states: every positive integer eventually reaches 1. No one has ever found a counterexample, and no one has proved it holds universally. It remains one of the most famous unsolved problems in mathematics.

In PGS, this is a protocol invariant — the runtime doesn't know what Collatz means. It simply traverses the declared execution graph. VIOLATION is a first-class exit, not a bug. The trace is verifiable proof of what was computed.
